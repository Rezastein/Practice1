## input
- Membuat variable
```sh
fullname
```
- membuat input pada variable 
```sh
fullname = input()
```
- membuat type data pada input
<br> Type data **string/text** adalah **str**
<br> Type data **integer/angka** adalah **int**
```sh
full name = str(input())
phonenumber = int(input())
```
